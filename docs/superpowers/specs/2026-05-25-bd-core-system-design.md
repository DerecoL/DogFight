# BD 核心体系设计

## 背景与目标

“BD”在本文中指 Build，也就是玩家围绕狗狗被动、装备、职业装备、遗物、药水、附魔、商店选择和摆位形成的构筑体系。

当前项目已经具备多条构筑线索：小点、大点、多重、蓄水、毒伤、护盾荆棘、吸血成长、爆鸣高频、天命、大物品和经济流。但这些线索主要散落在装备标签、职业装备和局内效果里，还缺少一层统一的体系定义。本文先把 BD 路线、组件角色、缺口和克制关系定型，为后续新增装备、遗物、职业装备、商店提示和数值平衡提供同一套判断标准。

BD 体系只作为识别、提示、统计和内容设计的参考层，不作为玩家构筑的硬限制。任何装备、遗物或狗狗都允许同时携带多个 BD 标签；系统不应用 BD 标签限制购买、穿戴、触发、商店刷新或混搭收益。

本阶段只产出设计文档，不修改源码类型、装备数值、战斗逻辑、UI 或外部 Excel 数值文档。

## 体系原则

第一阶段遵守四条自由构筑原则：

- 标签只描述倾向，不描述资格。`small`、`multi`、`poison` 等标签表示“这件东西可以服务某条路线”，不表示“只能给这条路线使用”。
- 主副 BD 只用于阅读和推荐，不用于锁职业、锁装备或锁商店池。
- 一件装备可以同时属于多条 BD，例如多重件既可以服务多重流，也可以服务爆鸣高频流或蓄水流。
- 缺口提示只能表达“你可能还缺某类组件”，不能表达“你必须补某件装备”或“当前构筑错误”。

每条 BD 都应尽量具备 6 类组件：

- 核心件：决定构筑方向，玩家拿到后会改变后续购买和摆位选择。
- 引擎件：提高触发频率、改骰、额外投掷、充能、映射触发或资源获得。
- 收益件：把频率或条件转化为伤害、治疗、护盾、控制、成长或经济收益。
- 防守件：提供治疗、护盾、减伤、净化、最大生命、控制免疫或拖时间能力。
- 反制件：针对其他 BD 的关键上限，例如反高频、反成长、反治疗、反毒、反控制、反大物品、反单核心点数。
- 过渡件：前中期能临时支撑路线，允许玩家在核心件没来之前不至于空转。

BD 的最低成型标准：

- 至少有 1 个核心件或职业被动能明确指向该路线。
- 至少有 1 个引擎件和 1 个收益件能形成正反馈。
- 至少有 1 个通用或职业来源能提供防守或反制。
- 玩家能在商店、装备标签或战斗反馈中看懂自己正在接近哪条路线。

## BD 路线总览

| BD 路线 | 当前状态 | 主要适配狗狗 | 核心目标 | 优先缺口 |
| --- | --- | --- | --- | --- |
| 小点流 | 半成型 | 柴犬、青蛙 | 围绕 1/2/3 高频触发和小型装备滚动收益 | 终局收益、反制说明 |
| 大点流 | 半成型 | 萨摩耶、恶霸 | 围绕 4/5/6 高伤、护盾、荆棘或大型物品 | 过渡件、反控制 |
| 多重流 | 半成型 | 土狗、柴犬 | 通过多重次数、相邻加成和重复触发堆频率 | 核心识别、反高频 |
| 蓄水流 | 已有职业骨架 | 青蛙 | 将显式点数装备改为计时充水，稳定触发持续收益 | 通用联动件、反制件 |
| 毒伤流 | 只有零件 | 柴犬、通用 | 通过中毒层数和毒结算加成造成持续伤害 | 核心件、防守件、反净化 |
| 护盾荆棘流 | 半成型 | 萨摩耶、狗皇帝 | 用护盾拖时间，用荆棘反伤和正面层数压制对手 | 成型奖励、反穿透 |
| 吸血成长流 | 半成型 | 通用、土狗 | 用成长伤害或高伤装备配合吸血形成续航上限 | 过渡件、反治疗 |
| 爆鸣高频流 | 半成型 | 土狗、柴犬、青蛙 | 通过大量成功触发积攒爆鸣计数打终局爆发 | 前期过渡、反高频 |
| 天命流 | 已有职业骨架 | 狗皇帝 | 围绕天命数字改点、强制命中和单点翻倍爆发 | 通用联动件、反单点 |
| 大物品流 | 已有职业骨架 | 恶霸、萨摩耶 | 用 4 格或被视作大型的装备打高质量触发 | 小件过渡、反大物品 |
| 经济流 | 只有零件 | 通用、恶霸 | 牺牲短期战力换金币、出售收益或品质速度 | 成型奖励、失败兜底 |

## 路线细化

### 小点流

定位：前中期频率路线，依赖 1/2/3 点触发、小型装备和柴犬改骰。

- 核心件：柴犬被动、极速太刀、忍法·影分身、忍法·破。
- 引擎件：燕回太刀、训练飞盘、夜巡犬探照灯、点金手·右、半截骰·右。
- 收益件：小型咬击、断裂的犬齿、跳蚤飞盘、磨牙成长剑。
- 防守件：耻辱圈、防咬套、特效狗薄荷、全自动饮水机。
- 反制件：逆毛净化梳可反正面层数，发酵的翻倒垃圾桶可失效最右侧装备。
- 过渡件：1/2/3 点牙咬、小型咬击、耻辱圈。
- 推荐补强：增加一个“低点触发次数越多收益越高”的通用终局件，使小点流不只依赖职业装备。

### 大点流

定位：中后期质量路线，依赖 4/5/6 点触发、高基础伤害、护盾和荆棘。

- 核心件：萨摩耶被动、松软毛皮、荆棘毛发、冰霜毛发、绝对零度。
- 引擎件：不畏严寒、点金手·左、半截骰·左、幸运狗尾草。
- 收益件：尖刺项圈、巨型骨棒、消防栓战斧、巨型恐龙腿骨、雪崩核心。
- 防守件：狗屋木板盾、不可侵犯的纯金狗窝、松软毛皮。
- 反制件：逆毛净化梳克制荆棘和护盾；无理强拆克制大型装备。
- 过渡件：4/5/6 点牙咬、尖刺项圈、狗屋木板盾。
- 推荐补强：补一个大点流的“低频但可靠过渡件”，避免核心没来时只剩普通高点伤害。

### 多重流

定位：频率转收益路线，依赖多重次数、重复触发和相邻光环。

- 核心件：荷叶之海、九州护腕、幸运爪垫、牛奶骨头、橡胶球、训练飞盘。
- 引擎件：土狗被动、咬尾巴、追车车、充能项圈、哈士奇永动机。
- 收益件：九州护腕、爆鸣计数器、磨牙成长剑、淬毒狗牙。
- 防守件：牛奶骨头、九州护腕提供护盾，老旧项圈提供最大生命成长。
- 反制件：需要后续补反高频件；现有失效类效果只能点状处理。
- 过渡件：牛奶骨头、橡胶球、训练飞盘。
- 推荐补强：给多重流增加明确路线标记和反制入口，例如“下次多重只触发一次”或“重复触发衰减”类反制。

### 蓄水流

定位：青蛙职业路线，把点数触发转为稳定计时触发，降低骰点波动。

- 核心件：青蛙被动、荷叶水泵、蛙鸣鼓、满池闸门。
- 引擎件：雨滴漏斗、莲池回声、暴雨季、胡萝卜、纸巾。
- 收益件：磨牙成长剑、爆鸣计数器、多重装备和任意显式点数装备。
- 防守件：全自动饮水机、狗屋木板盾、牛奶骨头。
- 反制件：当前缺少专门反蓄水或反计时充能工具。
- 过渡件：小点装备、稳定触发装备、低价治疗或护盾装备。
- 推荐补强：增加“延缓敌方充能/蓄水进度”或“打断最高进度装备”的反制方向。

### 毒伤流

定位：持续伤害路线，用中毒层数绕开普通直接伤害节奏。

- 核心件：忍法·剧毒、淬毒狗牙、发酵的翻倒垃圾桶、坏狗狗作案手册。
- 引擎件：柴犬小点频率、土狗额外投掷、跳蚤飞盘、毒相关小型装备。
- 收益件：中毒结算、坏狗狗作案手册的毒结算加伤。
- 防守件：毒伤流本身缺防守，通常依赖护盾、治疗或失效拖时间。
- 反制件：特效狗薄荷、狗皇帝金丝龙袍、纯金狗窝护盾期间负面免疫。
- 过渡件：跳蚤飞盘、小型咬击、断裂的犬齿。
- 推荐补强：补一个毒伤核心件，让毒流不只依赖柴犬终阶或单件被动。

### 护盾荆棘流

定位：防守反伤路线，用护盾和正面层数拖长战斗并惩罚攻击。

- 核心件：荆棘毛发、带刺防爆冲胸背、炸毛护颈圈、不可侵犯的纯金狗窝。
- 引擎件：大点流、萨摩耶被动、护盾装备、多重触发护盾件。
- 收益件：荆棘反伤、盾量维持、敌方攻击频率越高收益越高。
- 防守件：狗屋木板盾、纯金狗窝、松软毛皮、全自动饮水机。
- 反制件：逆毛净化梳、巨型恐龙腿骨、特效狗薄荷只能处理自身负面但不能反盾。
- 过渡件：耻辱圈、狗屋木板盾、带刺防爆冲胸背。
- 推荐补强：增加护盾荆棘的终局奖励，同时保留逆毛净化梳作为明确通用克制。

### 吸血成长流

定位：后期续航路线，把成长伤害或高额直接生命伤害转成治疗。

- 核心件：血契犬牙、磨牙成长剑、嗜血疯狗之牙。
- 引擎件：夜巡犬探照灯、土狗额外投掷、青蛙蓄水、柴犬小点频率。
- 收益件：成长剑后续伤害、血契赋予吸血、高伤装备实际生命伤害。
- 防守件：吸血本身、牛奶骨头、全自动饮水机、护盾装备。
- 反制件：当前缺少稳定反治疗手段；冰冻和失效只能间接降低触发。
- 过渡件：磨牙成长剑、嗜血疯狗之牙、普通治疗装备。
- 推荐补强：优先增加“治疗降低/吸血失效/过量治疗转弱点”的反治疗体系。

### 爆鸣高频流

定位：终局爆发路线，通过其他装备成功触发次数积累爆鸣计数。

- 核心件：爆鸣计数器。
- 引擎件：土狗被动、咬尾巴、追车车、燕回太刀、夜巡犬探照灯、青蛙蓄水、多重装备。
- 收益件：爆鸣计数器的 50 次触发爆发伤害。
- 防守件：需要外接治疗、护盾或控制，因为爆鸣本身不提供防守。
- 反制件：当前缺少专门反高频工具；冻结、失效和反增益只能部分压制。
- 过渡件：训练飞盘、牛奶骨头、橡胶球、低价多重装备。
- 推荐补强：增加反高频标识和道具，例如“敌方连续触发到阈值后失效一次”。

### 天命流

定位：狗皇帝职业路线，把随机骰点收束到单个天命数字，追求单点稳定和爆发。

- 核心件：狗皇帝被动、御用骰盅、垂帘听政、蛮横圣旨、亡国之君。
- 引擎件：御用骰盅保底、蛮横圣旨改点、垂帘听政把相邻装备纳入天命。
- 收益件：天命命中翻倍、亡国之君使天命装备生效 2 次。
- 防守件：权臣架空、金丝龙袍。
- 反制件：当前缺少反单核心点数工具；纸巾和胡萝卜可能间接干扰触发点。
- 过渡件：幸运爪垫、大点和小点通用装备、护盾装备。
- 推荐补强：增加“改变敌方核心数字/暂时错位指定点数”的反单点体系。

### 大物品流

定位：恶霸职业路线，通过大型物品、大体积收益和非大型联动打高质量触发。

- 核心件：恶霸被动、恶霸金库、恶霸健身房、恶霸袖标、无情献祭。
- 引擎件：大型物品触发非大型物品、小型物品触发大型物品、幸运狗尾草。
- 收益件：巨型骨棒、巨型恐龙腿骨、发酵垃圾桶、巨兽压迫。
- 防守件：小狗窝、纯金狗窝、狗屋木板盾。
- 反制件：无理强拆、发酵垃圾桶、逆毛净化梳。
- 过渡件：尖刺项圈、护卫背心、狗屋木板盾。
- 推荐补强：给大物品流补更清晰的小件过渡，不让路线过度依赖职业奖励。

### 经济流

定位：运营路线，用短期弱战力换后续品质、金币或装备数量优势。

- 核心件：狗狗金元宝、狗狗银元宝、恶霸金库。
- 引擎件：出售收益、战后获得大型物品、升级商店、药水商店。
- 收益件：更快升级核心件、更早补齐高价装备、更高出售回收。
- 防守件：当前缺少经济流专属防守，通常靠低价盾、治疗和过渡伤害。
- 反制件：经济流容易被快攻压制，不需要专门克制件，重点是失败兜底。
- 过渡件：银元宝、低价点牙咬、耻辱圈、牛奶骨头。
- 推荐补强：定义经济流的风险回报边界，例如“经济件占格但不触发”的可视化提示和局后收益预估。

## 狗狗与 BD 映射

| 狗狗 | 主 BD | 副 BD | 说明 |
| --- | --- | --- | --- |
| 柴犬 | 小点流 | 毒伤流、爆鸣高频流、多重流 | 小点改骰和小型职业装备天然支持高频触发，也能承接毒和爆鸣。 |
| 萨摩耶 | 大点流 | 护盾荆棘流、控制流 | 大点倾向与治疗、荆棘、虚弱、冻结强相关。 |
| 土狗 | 多重流 | 爆鸣高频流、吸血成长流 | 额外投掷和连锁投掷适合触发次数、成长伤害和爆鸣计数。 |
| 恶霸 | 大物品流 | 经济流、大点流 | 大型物品、战后拿装备和大件联动是核心身份。 |
| 狗皇帝 | 天命流 | 护盾防守流、单点爆发流 | 天命数字让装备选择围绕单点命中和改点展开。 |
| 青蛙 | 蓄水流 | 小点流、爆鸣高频流、多重流 | 蓄水把骰点变成稳定计时，适合承接各种显式点数装备。 |

覆盖结论：每个现有狗狗都至少能映射到 1 条主 BD 和 1 条副 BD。当前最需要补的不是职业身份，而是通用标签、路线识别、过渡件和反制件。

## 标签映射建议

后续实现构筑识别时，应优先复用现有 `tags`，不急于改战斗逻辑。标签映射必须是多对多关系：一个标签可以映射多条 BD，一条 BD 也可以由多个标签、职业装备、遗物或系统特征共同识别。

| 标签或特征 | 映射 BD | 组件倾向 |
| --- | --- | --- |
| `small`、触发点包含 1/2/3 | 小点流、蓄水流 | 引擎、过渡 |
| `big`、触发点包含 4/5/6 | 大点流、大物品流 | 收益、防守 |
| `multi` | 多重流、爆鸣高频流 | 引擎、收益 |
| `reservoir` | 蓄水流 | 核心、引擎 |
| `poison` | 毒伤流 | 核心、收益 |
| `shield`、`thorn`、`immune` | 护盾荆棘流 | 防守、收益 |
| `lifesteal`、`growth` | 吸血成长流 | 核心、收益 |
| `counter`、`trigger`、空点数但计数触发 | 爆鸣高频流 | 核心、反制 |
| `lucky` | 天命流 | 核心、引擎 |
| `large` 或尺寸 4 | 大物品流 | 核心、收益 |
| `economy`、`sell` | 经济流 | 核心、运营 |
| `cleanse`、`disable`、`shield-break` | 反制体系 | 反制、防守 |

## 克制关系

第一阶段不需要把克制关系全部做成新装备，但需要在设计和标签上明确方向：

- 反高频：克制多重流、爆鸣高频流、土狗连锁触发和青蛙稳定触发。后续可设计为连续触发阈值失效、重复触发衰减或触发冷却。
- 反成长：克制磨牙成长剑、雪崩核心和长期战斗上限。后续可设计为清空或压低单件战斗内成长。
- 反治疗：克制吸血成长流、萨摩耶治疗和经济拖后期。后续可设计为治疗降低、吸血失效或治疗转为弱点。
- 反毒：克制毒伤流。现有净化和护盾免疫已能承担一部分，但需要让毒流也有反净化或复毒手段。
- 反控制：克制冻结、失效、强制改点。后续可考虑短时免控或控制后返还收益。
- 反大物品：克制恶霸和高尺寸装备。现有无理强拆较明确，通用侧还可以补低价反大物品过渡。
- 反单核心点数：克制天命流和围绕单点堆叠的装备。后续可通过临时错位、指定数字扰动或触发点保护形成博弈。

## 第一阶段实施顺序

1. 标签与路线识别：建立 `BuildArchetype` 到现有 tags、装备 id、职业装备 id、遗物 id 的多对多映射表，所有映射均为非限制性标识。
2. 组件分层：给核心装备、职业装备和关键遗物标记 `BuildComponentRole`，先覆盖高价值路线，不追求一次性穷尽所有普通件。
3. 缺口标注：在文档或数据中标出每条路线缺核心、缺过渡、缺反制、缺防守或缺终局奖励。
4. 玩家提示原型：只做轻量提示，例如“当前偏向：小点/多重/爆鸣”，不做复杂推荐算法，也不提示“路线错误”。
5. 内容补强：根据缺口决定新增装备、遗物或职业奖励，并同步数值文档和构建产物。
